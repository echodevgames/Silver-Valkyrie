## Ice Trolls Mode

### Narrative

*"The Ice Trolls are a savage, territorial pair—bound by blood and fury. They've claimed this frozen pass as their hunting ground, and you're on the menu."*

### Qualification

Complete all 5 TROLL drop targets (T-R-O-L-L)

### Mode Start

- Drop targets reset and reform as ice barrier
- Ice Trolls appear behind the barrier (animated)
- Camera may briefly zoom to show them
- Voice: *"TONIGHT WE DINE ON VIKING SOUP!"*
- 30-second timer begins

### Objectives

1. **Clear the TROLL targets again** to access the saucer
2. **Hit the saucer** to wound the first troll
   - Troll 1: *"ARRRGH! You'll pay for that!"*
   - Targets partially reset (T-R-O only)
3. **Clear remaining targets** and hit saucer again
4. **Second saucer hit** kills the mate
   - Troll 2: *"NOOO! Sister!"*
   - Mode victory

### Failure Condition

Timer expires before second saucer hit

### Scoring

| Action | Points |
|--------|--------|
| TROLL target (during mode) | 15,000 |
| First saucer hit | 100,000 |
| Second saucer hit (victory) | 250,000 |
| Mode completion bonus | 500,000 |

### Visual Elements

- Ice formations spread across upper playfield
- Breath clouds from trolls
- Blood splatter on saucer hits
- Death animation: troll collapses, ice cracks

### Voice Lines

**Trolls (taunting)**:
- *"Person-meat! My favorite!"*
- *"Tiny warrior... big meal!"*
- *"We'll pick our teeth with your bones!"*

**Sasha (victory)**:
- *"Back to the pit!"*
- *"Thou shall curse this land no more!"*

## Giant Ogre Mode

### Narrative

*"The Ogre has terrorized these lands for generations. His skull is thick, but Sasha's blade is sharper."*

### Qualification

Hit the Ogre Head bash target 3 times in quick succession (within 5 seconds)

### Mode Start

- Ogre roars and becomes fully active
- Head animation intensifies
- Voice: *"I'LL CRUSH YOU LIKE A WALNUT!"*
- 45-second timer begins

### Objectives (Phase-Based)

**Phase 1: Stun the Ogre**
- Hit the bash target 5 more times
- Each hit: Ogre staggers, *"OOF!"* *"GAH!"* *"STOP THAT!"*
- After 5 hits: Ogre is stunned, mouth hangs open

**Phase 2: The Killing Blow**
- Shoot the ball INTO the ogre's open mouth (modified target)
- This is a precise shot—skill required
- Voice: *"NOW! STRIKE!"* (Sasha)

**Phase 3: Victory**
- Decapitation animation
- Ogre: *"NOOO! MY BEAUTIFUL FACE!"*
- Mode complete

### Failure Condition

Timer expires before killing blow lands

### Scoring

| Action | Points |
|--------|--------|
| Bash target (stun phase) | 50,000 |
| Stunning the Ogre | 100,000 |
| Killing blow | 500,000 |
| Mode completion bonus | 750,000 |
| **Brutality Bonus** | +250,000 (if completed with 20+ sec remaining) |

### Visual Elements

- Ogre head fully animated throughout
- Impact shake on every bash hit
- Drool and spit effects
- Stunned state: eyes spinning, tongue out
- Death: head rolls away (matches commissioned art)

### Voice Lines

**Ogre**:
- *"I'll crush you like a walnut!"*
- *"Stand still, little bug!"*
- *"My head... it's spinning..."*
- *"Noooo! My beautiful face!"*

**Sasha**:
- *"Face me, beast!"*
- *"Your reign ends NOW!"*

## Draugr Horde Mode

### Narrative

*"From the frozen earth they rise... restless, ravenous, and many. The undead warriors of ages past hunger for the living."*

### Qualification

Accumulate 50 pop bumper hits (tracked across entire game, persists between balls)

### Mode Start

- Draugr begin rising from the playfield (animated background)
- Pop bumpers glow ghostly blue
- Eerie choir music begins
- Voice: *"THE DEAD RISE!"*
- No timer—mode runs until ball drains

### Objectives

- **Survive** and rack up points in the bumper area
- Every bumper hit now worth 5x normal value
- Completing RUNE during this mode triggers **Rune Blast** (all bumpers flash, 10x for 5 seconds)
- Exit mode by shooting Longhouse scoop for **Horde Slayer Bonus**

### Failure Condition

None—mode ends naturally on drain. Points collected are kept.

### Scoring

| Action | Points |
|--------|--------|
| Pop bumper (during mode) | 5,000 |
| Pop bumper (Rune Blast) | 10,000 |
| RUNE completion bonus | 50,000 |
| Longhouse exit (Horde Slayer) | 100,000 × bumper hits during mode |

### Visual Elements

- Draugr skeletons rising from ground (background animation)
- Ghostly blue fog across bumper area
- Bumpers pulse with undead energy
- Skeletal hands reaching from below

### Voice Lines

**Ambient**:
- Moaning, rattling bones
- Distant battle cries

**Sasha**:
- *"You should have stayed buried!"*
- *"Return to your graves!"*

## Berserk Multiball

### Narrative

*"When the battle-fury takes hold, Sasha becomes an unstoppable force. Two blades. No mercy."*

### Qualification

Spell B-E-R-S-E-R-K by hitting designated targets:

| Letter | Target |
|--------|--------|
| B | Left orbit completion |
| E | Complete AXE drops |
| R | 10 pop bumper hits |
| S | Either SLAY inlane |
| E | Right orbit completion |
| R | Complete RUNE rollovers |
| K | Longhouse scoop |

Letters can be collected in any order. Progress persists between balls.

### Mode Start

1. Shoot Longhouse scoop with BERSERK complete
2. Ball captured
3. *"SASHA GOES BERSERK!"*
4. Second ball launches
5. Playfield flashes red
6. 30-second timer begins
7. **ALL SCORING DOUBLED**

### Objectives

- Keep both balls in play as long as possible
- All shots worth 2x
- No specific targets—just SCORE

### Failure Condition

- One ball drains: Multiball ends immediately, timer stops
- Both balls drain: Mode over

### Scoring

| Action | Points |
|--------|--------|
| All scoring | 2x normal value |
| Both balls active bonus | +10,000 per second |
| Multiball survival bonus | 25,000 × seconds survived |

### Visual Elements

- Red-tinted lighting across playfield
- Sasha dual-wielding animation on screen
- Motion blur on ball trails
- Aggressive particle effects on all hits

### Voice Lines

**Sasha**:
- *"UNLEASH THE FURY!"*
- *"DUAL STRIKE!"*
- *"FEEL MY WRATH!"*

**On drain**:
- *"The fury subsides..."*

## Feast Hurry-Up

### Narrative

*"Between battles, warriors gather in the longhouse. The mead flows, and glory awaits those who drink deep."*

### Qualification

Hit Longhouse scoop 5 times when NO other mode is active

### Mode Start

- Longhouse lights up golden
- Feast sounds (cheering, drinking)
- Hurry-up value displayed: starts at 500,000
- Value counts down rapidly

### Objectives

- Shoot the Longhouse scoop before timer expires
- Value decreases every second
- Minimum value: 50,000

### Failure Condition

### Scoring

### Visual Elements

- Golden glow on Longhouse
- Countdown numbers prominently displayed
- Mead mugs and feast imagery
- Cheering Vikings in background

### Voice Lines

**Ambient**:
- Cheering, toasting
- *"SKÅL!"*

### Narrative

*"You have defeated all who stood before you. Now, the gates of Valhalla open—but only the worthy may enter. The spirits of your fallen foes return for one final battle."*

### Qualification

Complete ALL five previous modes in a single game:
- [x] Ice Trolls
- [x] Giant Ogre  
- [x] Draugr Horde
- [x] Berserk Multiball

## Mode Interaction Rules

### Stacking

- **No stacking**: Only one mode active at a time (except wizard mode)
- **Multiball exception**: Berserk can run alongside qualified-but-not-started modes
- **Wizard mode**: Supersedes everything

### Priority

1. Master Slayer Mode (if qualified)
2. Berserk Multiball (if active)
3. Most recently qualified mode
4. Feast Hurry-Up (lowest priority)

### Persistence

- Mode COMPLETION persists between balls (for wizard mode tracking)
- Mode PROGRESS resets on ball drain (except Draugr bumper count)
- BERSERK letters persist between balls
- SLAY multiplier resets each ball

---

## Mode Select (Optional Feature)

If multiple modes are qualified but not started, player can choose:

- Shoot LEFT orbit: Start left-most qualified mode
- Shoot RIGHT orbit: Start right-most qualified mode
- Shoot LONGHOUSE: Start highest-value qualified mode

This prevents mode qualification from feeling "wasted" and adds player agency.
